超好用的WOW.js+Animate

有的页面在向下滚动的时候，有些元素会产生细小的动画效果。虽然动画比较小，但却能吸引你的注意。如果你希望你的页面也更加有趣，那么你可以试试 WOW.js。

WOW.js 依赖 animate.css，所以它支持 animate.css 多达 60 多种的动画效果，能满足您的各种需求。

使用方法

引入依赖

animate.css & wow.js

如何使用

在javascript中进行初始化

在使用对象添加 class

可用属性

  1. data-wow-duration（动画持续时间）

    data-wow-duration="2s"

  2. data-wow-delay（动画延迟时间）

    data-wow-delay="5s"

  3. data-wow-offset（偏移量）距离可视区域多远开始执行动画

    data-wow-offset="10"

  4. data-wow-iteration=”10”(重复次数)

    data-wow-iteration="10"

可自定义配置

如果需要自定义配置，可如下使用：

配置

属性/方法 类型 默认值 说明
boxClass 字符串 ‘wow’ 需要执行动画的元素的 class
animateClass 字符串 ‘animated’ animation.css 动画的 class
offset 整数 0 距离可视区域多少开始执行动画
mobile 布尔值 true 双方在移动设备上执行动画
live 布尔值 true 异步加载的内容是否有效

animate.css
